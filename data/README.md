# 데이터 설명

이 폴더에는 모델 학습에 사용되는 모든 데이터가 들어 있습니다.  
폴더 세 개로 나뉘며, 각각 역할이 다릅니다.

```
data/
├── raw/          원본 수집 데이터 (용량 문제로 저장소에 포함되지 않음)
├── processed/    모델 학습에 바로 쓸 수 있는 형태로 가공된 데이터
└── splits/       학습 / 검증 / 테스트 분할 정보 및 AI 응답 원문
```

---

## 데이터가 만들어진 과정

```
폼클렌저 300개 상품 페이지 수집
          ↓
각 페이지에서 수치 특징 24가지 추출  →  raw/page_features.csv
          ↓
상품 3개씩 묶어 AI에게 순위 요청  →  raw/response.csv
          ↓
비교 세트별로 정리  →  processed/trial_items.csv
          ↓
학습 / 검증 / 테스트로 분할  →  processed/step1_*_features.csv
                                processed/step2_*_features.csv
```

---

## raw/ — 원본 수집 데이터

> 이 폴더는 용량이 너무 커서(111MB 이상) 저장소에 포함되지 않습니다.

| 파일 | 내용 |
|------|------|
| page_features.csv | 폼클렌저 상품 페이지 300개에서 추출한 수치 특징 원본. 한 행이 상품 한 개. |
| response.csv | AI에게 상품 3개씩 비교 요청을 보냈을 때 받은 응답 원문. 순위 판단의 근거가 됨. |
| full_prompt.csv | AI에게 보낸 요청 전문. |

---

## processed/ — 학습용으로 가공된 데이터

### trial_items.csv — 비교 세트별 상품 특징

모든 비교 세트에 등장한 상품의 특징을 정리한 기준 테이블입니다.  
한 비교 세트에 상품 3개가 등장하므로, 한 비교 세트 번호당 행이 3개씩 있습니다.

**행 구성 예시:**

| 비교 세트 번호 | 제시 위치 | 브랜드 | 가격 | 텍스트 길이 | 이미지 수 | ... |
|---|---|---|---|---|---|---|
| main_00001 | 1번째로 제시 | 더후 | 43,000원 | 15,498자 | 88개 | ... |
| main_00001 | 2번째로 제시 | 에뛰드 | 8,000원 | 4,288자 | 22개 | ... |
| main_00001 | 3번째로 제시 | 아로마티카 | 20,000원 | 21,081자 | 88개 | ... |

**포함된 수치 특징 24가지:**

| 분류 | 항목 |
|------|------|
| 텍스트 구성 | 전체 텍스트 길이, 이미지 수, 표 수, 목록 항목 수, 단락 수, 섹션 수 |
| 신뢰도 표현 | 구조화 데이터 항목 수, 구체적 숫자 표현 수, 모호한 표현 수, 수치 구체성 비율 |
| 가격 / 용량 | 가격(원), 용량(ml) |
| 성분 / 효능 | 주요 성분 수, pH 수치, 효능 키워드 수, 텍스처 키워드 수, 부정적 표현 수 |
| 피부 타입 | 대상 피부 타입 수 |
| 리뷰 | 평점, 리뷰 수 |
| 인증 | 인증 마크 수 |
| 소셜 증거 | 소셜 증거 요소 수, 외부 기관 인용 수 |
| 종합 점수 | AI 친화도 종합 점수 |

---

### step1\_\*\_features.csv / step2\_\*\_features.csv — 학습 입력 데이터

trial_items.csv에 **AI가 매긴 순위**를 결합한 파일입니다. 모델이 실제로 학습에 사용하는 데이터입니다.  
각각 학습용(`train`), 검증용(`val`), 테스트용(`test`)으로 나뉩니다.

trial_items.csv와의 차이점: `AI 순위` 열과 `AI 엔진 종류`, `실험 회차` 열이 추가되어 있습니다.

**분할 방식 두 가지:**

| | step1 | step2 |
|---|---|---|
| 분할 기준 | 비교 세트 단위로 무작위 분할 | 특정 브랜드를 테스트 전용으로 사전 지정 |
| 테스트 상품 | 학습에서도 본 상품 (새로운 조합으로 등장) | 학습에서 단 한 번도 보지 않은 새 브랜드 |
| 평가 목적 | 기본 순위 예측 성능 확인 | 새 상품에 대한 일반화 능력 확인 |
| 학습 세트 크기 | 3,709개 비교 세트 | 3,174개 비교 세트 |
| 검증 세트 크기 | 810개 비교 세트 | 561개 비교 세트 |
| 테스트 세트 크기 | 807개 비교 세트 | 1,563개 비교 세트 |

step2의 테스트 전용 브랜드 11개: 김소형풀, 넘버즈인, 믹순, 비욘드, 수이스킨, 숨37, 아로마티카, 아크웰, 아페리레, 제이준, 피캄

## splits/ — 분할 정보 및 AI 응답 상세

### step1\_\*\.csv / step2\_\*\.csv — 비교 세트 메타데이터

각 비교 세트에 대한 상세 정보를 담고 있습니다. 모델 학습에 직접 사용되지는 않으며, 분석과 검증에 활용합니다.

| 열 | 의미 |
|---|---|
| 비교 세트 번호 | 비교 실험의 고유 번호 |
| 실험 회차 | 동일한 세트를 여러 번 실험한 경우 회차 구분 |
| AI 엔진 | 순위를 매긴 AI 종류 (예: OpenAI GPT) |
| 제시 상품 1·2·3 | 비교에 사용된 세 상품의 브랜드명 |
| 1위 브랜드 / 제품 유형 / 가격 | AI가 1위로 선택한 상품 정보 |
| 2위 / 3위 정보 | AI가 2위, 3위로 선택한 상품 정보 |
| 숫자 표현 수 / 밀도 | AI 응답에 등장한 구체적 수치 표현 |
| 안전성 / 피부 타입 키워드 수 | AI 응답에서 언급된 안전성·피부 관련 단어 수 |
| 응답 길이 | AI 응답의 글자 수 |
| API 비용 | 해당 요청에 소요된 비용 |
| 응답 요약 | AI 응답의 앞부분 발췌 |

### split_meta.json — 분할 설정 요약

어떤 기준으로 학습 / 검증 / 테스트를 나눴는지 기록한 파일입니다.

```json
{
  "seed": 42,                         ← 재현 가능하도록 고정한 무작위 시드
  "holdout_brands": ["아로마티카", ...], ← step2 테스트 전용으로 제외한 브랜드 목록
  "set_id_counts": { ... },           ← 각 분할의 비교 세트 수
  "description": { ... }              ← 분할 방식 설명
}
```

## 한눈에 보는 데이터 흐름

```
[원본 상품 페이지]
      ↓ 수치 특징 추출
[raw/page_features.csv]
      ↓ 3개씩 묶어 AI에게 순위 요청
[raw/response.csv]
      ↓ 비교 세트 단위로 정리
[processed/trial_items.csv]  ←  상품 특징 + 어떤 조합으로 비교됐는지
      ↓ AI 순위 결합 + 학습/검증/테스트 분할
[processed/step1_train_features.csv]  ←  모델이 실제 학습하는 데이터
[processed/step1_val_features.csv]
[processed/step1_test_features.csv]
[processed/step2_*.csv]              ←  새 브랜드 일반화 테스트용
```
